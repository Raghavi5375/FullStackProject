HTML Classes

The class attribute is often used to point to a class name in a style sheet. We can essentially point to the HTML in our css to manipulate it and make it look better. It can also be used by a JavaScript with backend code, with the specific class name. Notice in the CSS how we can reference the class name using .country. This is going to style the div block where "country" is the class. In this example, we have used a class in order to apply CSS styling to multiple html blocks. Be sure to click both the HTML and CSS tabs.

HTML ID's

An HTML id attribute is used to specify a unique id for an HTML element.

You cannot have more than one element with the same id in an HTML document. Note the difference between an id and a class, as you can have multiple html elements with the same class name. However, you can only have one element with a specific id in the html.
