What is CSS?

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

Styling html with a CSS id selector

We can also use ID's as selectors in our CSS. Notice that the only html elements the CSS styling is applied to is the div tag where 'brazil' is the id.

Styling html with a CSS class selector

We can also style html using a class selector in CSS. Class selectors must be preceded by ".".
Notice that we can specify multiple classes in the html as opposed to id.
When we style for a CSS class selector, the only html elements that are styled are those with the class selector name.
